# ASN2 - Airbnb Data Viewer

A Node.js and Express application for viewing and searching Airbnb property data.

## Features
- **View Data**: Browse a list of Airbnb properties.
- **Search**:
  - Search by Property Name
  - Search by Property ID
  - Filter by Price Range
- **Clean Data View**: View data with empty fields filtered out.
- **RESTful Routes**: Clean URL structure.

## Prerequisites
- Node.js installed
- npm (Node Package Manager)

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Divjit07/ASN2.git
    cd ASN2
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

## Running the Application

1.  Start the development server:
    ```bash
    npm run dev
    ```

2.  Open your browser and navigate to:
    [http://localhost:3000](http://localhost:3000)

## Project Structure
- `app.js`: Main application entry point.
- `views/`: Handlebars templates for the UI.
- `public/`: Static assets (CSS, images).
- `data/`: JSON data files.
